クラス 一覧

ArrayExtensions

System.Arrayの機能を拡張します。 このクラスは静的です。

ConsoleUtil

標準入出力ストリームをより便利に扱う為の機能を提供します。 このクラスは静的です。

DisposableBase

破棄可能なオブジェクトの基底クラスです。

SecureStringExtensions

System.Security.SecureStringクラスを拡張します。 このクラスは静的です。

SerializationInfoExtensions

System.Runtime.Serialization.SerializationInfoクラスを拡張します。 このクラスは静的です。

StringExtensions

System.Stringクラスを拡張します。 このクラスは静的です。

StringUtil

文字列をより便利に扱う為の機能を提供します。 このクラスは静的です。

UtilityConfigurationExtensions

IConfigurationの機能を拡張します。 このクラスは静的です。

UtilityContextExtensions

IContextの機能を拡張します。 このクラスは静的です。

待機可能な関数の戻り値を提供します。 このインターフェースを実装したクラスは非同期操作を表します。

IAwaiter<T>

非同期操作を待機しその実行結果を提供します。

IUtilityService
